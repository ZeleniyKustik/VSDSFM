В схеме есть 2 типа процессоров
1 тип процессор нужен для управления самой схемой. Минимальное и максимальное количество = 1
2 тип процессора ядра. Нужны для чтения и записи данных. Минимальное количество = 1, максимальное количество определяется исходя из размера памяти.

Примеры блока хранилища (Блок памяти, ячейка памяти или какая либо память из модов)

Есть 2 типа памяти.
1 тип порт через который идёт общение со схемой.
2 тип память в которой хранятся ваши даннные.

В схеме единый тип блоков хранилища как для порта так и для обычной памяти

Все процессоры соединяются в одинаковом порядке
Процессор нужно подключить к порту так чтобы через get link порт был под подключением номер 1.
Затем нужно подключить все остальные блоки хранилища

У каждого ядра в коде должен быть свой уникальный id они начинаются от 0 и каждое следующее id увеличивается на 1


После всего этого можно переходить к общению со схемой через порт с помощью API
